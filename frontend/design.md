# Design Notes

## UI Mockup

Elevator 1: status
Elevator 2: status
Elevator 3: status
...
Elevator n: status

Status line displays floor, movement, doors, alarm statuses.

Movement Status:
1. up
2. down
3. immobile

Doors Status:
1. closed
2. opened

Alarm Status:
1. On
2. Off

Clicking on elevator brings up an action dialog box allowing to request status changes.
